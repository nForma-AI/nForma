# Plan v0.36-02-01: Focus Filter Propagation & Missing-File Residuals

## Goal

Make the solve diagnostic engine report accurate residuals by (a) propagating the `focusSet` filter to all 19 layers and (b) returning `residual: -1` when required files are missing instead of silently returning 0.

## must_haves

1. Running `nf:solve --focus AUTH` propagates the focus filter to all 19 layers — every layer either filters its output to the focus set or marks results with `scoped: false`
2. When `check-results.ndjson` or `requirements.json` is missing, affected layers report `residual: -1` (unknown) instead of `residual: 0` — preventing false convergence signals

## Tasks

<task id="03">
<title>Change missing-file returns from residual 0 to -1</title>
<context>
Multiple sweep functions return `residual: 0` when required input files (requirements.json, check-results.ndjson, doc files) are missing. This hides unknown state as false convergence. The design requires `residual: -1` for missing-file scenarios.

`computeResidual()` already handles -1 correctly: `(layer.residual >= 0 ? layer.residual : 0)` — so changing these to -1 won't break the total computation.

Exception: `sweepTtoC()` with `runner=none` is an intentional config skip, NOT a missing file — leave as `residual: 0`.
</context>
<action>
Change `residual: 0` to `residual: -1` in these specific locations in `bin/nf-solve.cjs`:

1. **sweepFtoC** line ~1093: `run-formal-verify.cjs not found` → `residual: -1`
2. **sweepFtoC** line ~1134: `No check-results.ndjson generated` → `residual: -1`
3. **sweepRtoD** line ~1270: `requirements.json not found` → `residual: -1`
4. **sweepRtoD** line ~1280: `requirements.json parse error` → `residual: -1`
5. **sweepRtoD** line ~1294: `no doc files found` → `residual: -1`
6. **sweepDtoC** line ~1383: `no doc files found` → `residual: -1`
7. **sweepCtoR** line ~1647: `requirements.json not found` → `residual: -1`
8. **sweepCtoR** line ~1654: `requirements.json parse error` → `residual: -1`
9. **sweepDtoR** line ~1860: `requirements.json not found` → `residual: -1`
10. **sweepDtoR** line ~1867: `requirements.json parse error` → `residual: -1`
