**2. Create `bin/debate-formatter.cjs`:**

A CommonJS script that exports `extractFrontmatter(content)` and `validateDebate(content)`.

- `extractFrontmatter(content)`: Uses regex `/^---\n([\s\S]*?)\n---/` to extract YAML frontmatter. Parses key:value pairs. Handles array fields (`participants`, `tags`) in `[a, b, c]` format. Handles numeric fields (`rounds`). Handles quoted string values (strip quotes). Returns null if no frontmatter found.

- `validateDebate(content)`: Calls extractFrontmatter, checks that required fields `date` and `question` are present. Returns `{ valid: true, frontmatter }` or `{ valid: false, errors: [...] }`. All other fields optional.

- CLI mode: When run directly (`if (require.main === module)`), reads a file path from argv[1], runs validateDebate, prints JSON result to stdout, exits 0 for valid, 1 for invalid.

Use `'use strict'` at top. No external dependencies -- regex-based frontmatter parsing only. Follow project CommonJS patterns.

**3. Create `bin/debate-formatter.test.cjs`:**

Test cases using vitest (matching project test patterns):

1. **extractFrontmatter returns parsed frontmatter** -- Content with valid frontmatter returns object with date, question, consensus, rounds (as number), participants (as array), tags (as array).
2. **extractFrontmatter returns null for no frontmatter** -- Content without `---` delimiters returns null.
3. **extractFrontmatter handles partial frontmatter** -- Content with only date and question (no optional fields) returns object with those two fields.
4. **extractFrontmatter handles quoted values** -- `question: "foo bar"` returns `"foo bar"` without quotes.
5. **validateDebate returns valid for complete debate** -- Full template content returns `{ valid: true, frontmatter: {...} }`.
6. **validateDebate returns invalid for missing date** -- Content with frontmatter but no date field returns `{ valid: false, errors: [...] }`.
7. **validateDebate returns invalid for missing question** -- Content with frontmatter but no question field returns errors.
8. **validateDebate returns invalid for no frontmatter** -- Plain markdown returns errors.
