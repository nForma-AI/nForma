Create `bin/security-sweep.cjs` with the following exports:

1. `SECRET_PATTERNS` -- array of pattern objects:
   ```javascript
   const SECRET_PATTERNS = [
     { name: 'AWS Access Key', pattern: /AKIA[A-Z0-9]{16}/, severity: 'high' },
     { name: 'GitHub Token', pattern: /gh[ps]_[A-Za-z0-9_]{36,}/, severity: 'high' },
     { name: 'GitHub PAT', pattern: /github_pat_[A-Za-z0-9_]{22,}/, severity: 'high' },
     { name: 'Stripe Live Key', pattern: /[sr]k_live_[A-Za-z0-9]{20,}/, severity: 'high' },
     { name: 'OpenAI Key', pattern: /sk-[A-Za-z0-9]{32,}/, severity: 'high' },
     { name: 'Generic API Key Assignment', pattern: /(?:api[_-]?key|apikey)\s*[:=]\s*['"][^'"]{10,}['"]/i, severity: 'medium' },
     { name: 'Generic Secret Assignment', pattern: /(?:secret|password)\s*[:=]\s*['"][^'"]{8,}['"]/i, severity: 'medium' },
     { name: 'Debugger Statement', pattern: /^\s*debugger\s*;?\s*$/, severity: 'low' },
     { name: 'Sensitive Console.log', pattern: /console\.log\(.*(?:password|secret|token|key|api_key)/i, severity: 'low' },
   ];
   ```

2. `scanFile(filePath, content)` -- scans content line by line against SECRET_PATTERNS. Returns array of `{ file, line, column, pattern_name, severity, match }`. Skip lines containing common test indicator words: `test`, `mock`, `fake`, `example`, `dummy`, `fixture`, `placeholder`, `TODO`.

3. `scanDirectory(cwd, options)` -- options: `{ excludePatterns: [glob], maxFiles: 500 }`.
   - Gets tracked files via `git ls-files` (spawnSync, 5s timeout).
   - Excludes files matching: `*.test.*`, `*.spec.*`, `security-sweep.cjs` (self), `*.md`, `*.json`, `*.jsonl`, `node_modules/**`, `.planning/.quorum-cache/**`, `package-lock.json`.
   - Reads each file, calls scanFile(). Skips binary files (check for null bytes in first 512 bytes).
   - Returns `{ findings: [...], files_scanned: N, duration_ms: N }`.

4. `formatReport(scanResult)` -- formats findings into a markdown string for VERIFICATION.md:
   ```
   ## Security Sweep

   **Scanned:** N files in Nms
   **Findings:** N (H high, M medium, L low)

   | Severity | File | Line | Pattern | Match |
   |----------|------|------|---------|-------|
   | high | src/foo.js | 42 | AWS Access Key | AKIA... |

   _Advisory: Review findings and confirm whether they are genuine secrets or false positives._
   ```
   If no findings, output:
   ```
   ## Security Sweep

   **Scanned:** N files in Nms
   **Findings:** 0

   No hardcoded secrets, debug artifacts, or API keys detected.
   ```

5. CLI mode: when run as `node bin/security-sweep.cjs [--json] [--cwd /path]`:
   - Runs scanDirectory on the given cwd (default: process.cwd()).
   - With `--json`: outputs JSON result to stdout.
   - Without `--json`: outputs the formatted markdown report to stdout.
   - Logs a conformance event with action `security_sweep`.
   - Exit code 0 always (advisory, never blocks).

Update `hooks/conformance-schema.cjs` and `bin/conformance-schema.cjs`:
- Add `'security_sweep'` to VALID_ACTIONS array.

Use only node:fs, node:path, node:child_process. No external dependencies.
