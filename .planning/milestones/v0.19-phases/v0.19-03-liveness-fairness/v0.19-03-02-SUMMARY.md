## Key Decisions Made

- **Multi-name PROPERTIES handling:** Regex `m[1].trim().split(/\s+/).filter(n => /^\w+$/.test(n))` handles both `PROPERTY Foo` and `PROPERTIES Foo Bar` syntax
- **Unknown surface fail-open:** If a config name is not in SURFACE_MAP, return `[]` ‚Äî do not block unknown configs. This allows future surfaces to be added without breaking the gate.
- **specDir injection parameter:** Third parameter to `detectLivenessProperties` allows test isolation via tmp dirs instead of mocking fs
- **MCsafety test note:** MCsafety.cfg does have PROPERTY lines (AllTransitionsValid, DeliberationMonotone) but MCsafety is not in SURFACE_MAP, so function returns `[]` per unknown-surface rule. Test comment updated to reflect actual behavior.
