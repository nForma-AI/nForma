### Wiring Pattern Applied

```javascript
const { writeCheckResult } = require('./write-check-result.cjs');
// ... at every process.exit() call:
try { writeCheckResult({ tool: '<name>', formalism: '<type>', result: 'pass'|'fail', metadata: {} }); } catch (e) { process.stderr.write('[<name>] Warning: failed to write check result: ' + e.message + '\n'); }
process.exit(...);
```

All writes are wrapped in try/catch so runner exit behavior is unaffected if NDJSON write fails.
