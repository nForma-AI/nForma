## Known Limitations & Assumptions

3. **Numeric Threshold Detection**: Only checks if extractFormalExpected returns a number (not a heuristic)
4. **Dedup Threshold**: Levenshtein default 0.85; no adaptive tuning based on source type
5. **MCP Instruction Execution**: observe.md Step 4b requires direct MCP tool access in Claude context (not portable to non-Claude environments)
