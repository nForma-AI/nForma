## Decisions Made

- **Session context capped at 500 chars**: Prevents crowding other injections (quorum instructions, thinking budget, circuit breaker recovery)
- **Extract only "## Current Position" + "### Decisions"**: Focused extraction avoids noise, 20 line cap ensures conciseness
- **DEBUG_FIX_REGEX with word boundaries**: Avoids matching "debugger" or "fixed" — only match complete words (fix\b pattern)
- **Mutual exclusion: debug/fix > edit > new-feature**: Ordered checks prevent misclassification (edit prompt that mentions a fix gets root cause reasoning, not edit constraint)
- **All blocks use instructions += pattern**: Never write to stdout before cmdPattern check — fail-open is silent, quorum gate preserved
- **Sentinel flags are session-scoped**: File named nf-session-seen-{sessionId}.flag prevents delivery across sessions (SESSION-02 idempotency)
- **Fail-open with empty catch blocks**: Missing STATE.md, regex errors, flag write failures are silently skipped — hook continues to quorum injection
