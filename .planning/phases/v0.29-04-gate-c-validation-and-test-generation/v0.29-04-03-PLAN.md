  <action>
Add three new sweep functions to nf-solve.cjs and integrate them into computeResidual().

**sweepL1toL2():**
- Spawn `gate-a-grounding.cjs --json` via existing spawnTool pattern
- Parse JSON output: extract grounding_score, unexplained count breakdown (instrumentation_bug, model_gap, genuine_violation)
- Normalized residual: `Math.ceil((1 - grounding_score) * 10)` — yields 0-10 scale
- Detail: { grounding_score, target: 0.8, gap: target - score, unexplained_breakdown }
- If spawn fails: return { residual: -1, detail: { error: true, stderr } }

**sweepL2toL3():**
- Spawn `gate-b-abstraction.cjs --json` via spawnTool
- Parse JSON output: extract gate_b_score, orphaned count
- Additionally: load observed-fsm.json (dynamically, NOT hardcoded state-event pairs) and hazard-model.json; count L2 state-event pairs NOT covered by any L3 hazard
- Normalized residual: `Math.ceil((1 - gate_b_score) * 10) + Math.min(uncovered_pairs, 10)`
- Detail: { gate_b_score, uncovered_l2_pairs, orphaned_entries }
- If spawn fails: return { residual: -1, detail: { error: true } }

**sweepL3toTC():**
- First check if test-recipes.json exists. If stale (older than failure-mode-catalog.json by mtime comparison), spawn `test-recipe-gen.cjs` to regenerate before scoring.
- Spawn `gate-c-validation.cjs --json` via spawnTool
- Parse JSON output: extract gate_c_score, unvalidated count
- Normalized residual: `Math.ceil((1 - gate_c_score) * 10)`
- Detail: { gate_c_score, unvalidated_count, total_failure_modes, total_recipes }

**Integration into computeResidual():**
- Call the three sweep functions after existing reverse sweeps
- Add results to return object as `l1_to_l2`, `l2_to_l3`, `l3_to_tc`
- Add a new `layer_total` field: sum of layer sweep residuals (matching research recommendation for backward compatibility — do NOT add to existing `total`)
- Add layer sweeps to `health` indicator computation: if any layer residual > 5, health = "degraded"
- Skip layer sweeps in `--fast` mode (return { residual: -1, detail: { skipped: true, reason: 'fast mode' } })

**Update module.exports** to include sweepL1toL2, sweepL2toL3, sweepL3toTC.

**Update formatReport()** to include a "Layer Alignment" section showing L1->L2, L2->L3, L3->TC scores and layer_total.

**Update formatJSON()** to include l1_to_l2, l2_to_l3, l3_to_tc, layer_total in output.

**Update file header comment** to document the new 11 forward + 3 reverse + 3 layer sweeps.
