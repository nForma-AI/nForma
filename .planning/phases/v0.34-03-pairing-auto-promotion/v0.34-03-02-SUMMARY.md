### Clean Session Criteria

A solve session is "clean" when ALL of:
- Gate A wiring_evidence_score >= 1.0
- Gate B wiring_purpose_score >= 1.0
- Gate C wiring_coverage_score >= 1.0
- All semantic_scores >= 0.8
- No counterexamples in check-results.ndjson
