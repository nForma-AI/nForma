(Note: LAYER_KEYS is re-exported by oscillation-detector.cjs from solve-trend-helpers.cjs)

New exported functions:

1. **fitExponentialDecay(values)** (TRACK-03 core math):
   - Input: array of residual values (one per session, in chronological order)
   - Filter out values <= 0 (can't log-transform; -1 means skipped layer per v0.33-01 convention)
   - If valid.length < 10: return { status: 'INSUFFICIENT_DATA' }
   - If all valid values are identical: return { status: 'STABLE', lambda: 0, sessions_to_convergence: null }
   - Linearize: ln(y_i) = ln(a) - lambda * t_i (assume convergence target c = 0)
   - OLS on (t_i, ln(y_i)): slope = -lambda, intercept = ln(a)
   - Guard degenerate denominator (|denom| < 1e-12): return { status: 'DEGENERATE' }
   - If lambda <= 0: return { status: 'NOT_CONVERGING', lambda, a }
   - If current residual (last value in original array) <= threshold (0.5 default): return { status: 'CONVERGED', lambda, a, sessions_to_convergence: 0 }
   - Otherwise: sessions_to_convergence = ceil(ln(current / 0.5) / lambda)
   - Return { status: 'CONVERGING', lambda (6 decimals), a (4 decimals), sessions_to_convergence }
   - Use RESEARCH.md Example 2 code as implementation reference

2. **computeConvergenceVelocity(trendPath, opts)** (TRACK-03 per-layer):
   - Read trend entries via readTrendWindow(trendPath, opts.windowSize || 30)
   - For each layer in LAYER_KEYS:
     - Extract residual values from entries (entry.layers[layerKey])
     - Filter out -1 values (skipped layers per v0.33-01 fast-mode convention)
     - Call fitExponentialDecay(values)
   - Return object keyed by layer name: { status, lambda, a, sessions_to_convergence }

3. **updatePredictivePower(opts)** (main orchestration):
   - opts: { root } (project root path)
   - Resolve paths: debt.json, model-registry.json, per-model-gates.json, bug-to-property.json, solve-trend.jsonl (all under root/.planning/formal/)
   - Call linkBugsToProperties(debtPath, registryPath) -- write result to bug-to-property.json via writeBugToProperty()
   - Call computePerModelRecall(mappings, perModelGatesPath)
   - Call computeConvergenceVelocity(trendPath)
   - Return combined result: { linking: mappingObj, recall: recallScores, velocity: velocityByLayer }
   - Wrap entire body in try/catch -- fail-open with stderr warning (match oscillation-detector.cjs pattern)

4. **formatPredictivePowerSummary(results)** -- for solve report:
   - Combine recall summary + velocity summary into compact section
   - Velocity: show each layer with status and sessions_to_convergence (if applicable)
   - Keep total output under 25 lines
   - Return string prefixed with "--- Predictive Power ---"

Update module.exports to include all new functions: fitExponentialDecay, computeConvergenceVelocity, updatePredictivePower, formatPredictivePowerSummary.
