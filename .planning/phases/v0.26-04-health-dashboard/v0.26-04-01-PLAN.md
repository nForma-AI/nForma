8. `DASH-02: buildDashboardLines does NOT show [stale] warning when lastUpdated is within 60 seconds` — Call `buildDashboardLines([], {}, {}, Date.now() - 10_000)`. Assert output does NOT include `'stale'`.

**DASH-03 tests (clean exit):**

9. `DASH-03: liveDashboard non-TTY fallback resolves without throwing (clean exit path)` — Call `liveDashboard()` after checking `if (process.stdout.isTTY) return;`. Assert `doesNotReject(liveDashboard())`. This tests the non-interactive path exits cleanly.

10. `DASH-03: liveDashboard non-TTY path produces output containing dashboard content` — Capture stdout during `liveDashboard()` call (use `process.stdout.write` spy or check return). Guard with `if (process.stdout.isTTY) return;`.
