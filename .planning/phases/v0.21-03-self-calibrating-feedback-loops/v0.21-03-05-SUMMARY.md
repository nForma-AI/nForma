## Key Files

### Created
- `bin/propose-debug-invariants.cjs` (165 lines) — Three mining heuristics:
  1. State transitions (`A -> B` or `A → B` patterns) → `PROPERTY TransitionHint_A_to_B`
  2. Root cause lines (`root_cause: <text>`) → `INVARIANT RootCauseHint_<sanitized>`
  3. Invariant candidates (`Invariant candidate: <text>`) → `PROPERTY InvariantCandidate_N`
  - `--non-interactive` mode: prints all candidates, exits 0 (CI safe)
  - Interactive mode: presents each candidate with accept/reject/skip
  - Accepted candidates written via `accept-debug-invariant.cjs` with valid session ID format
  - Fail-open: exits 0 when artifact absent or no candidates found
