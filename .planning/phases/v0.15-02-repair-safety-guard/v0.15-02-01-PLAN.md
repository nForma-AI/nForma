<objective>
Add a content-length safety gate to the `validate health --repair` command in `gsd-tools.cjs` so that `regenerateState` cannot silently overwrite a STATE.md with more than 50 lines without an explicit `--force` flag.

Purpose: Prevent the real data loss incident that occurred when `--repair` replaced the full QGSD STATE.md (160 lines) with a 15-line blank template. After this fix, `--repair` on a rich STATE.md prints a line-count warning in `repairs_performed` and exits without touching the file unless `--force` is also passed.

Output: 3 new SAFE-01 tests in `gsd-tools.test.cjs` (Wave 0 scaffold — TC-01 must be RED before the fix), 2 targeted changes to `gsd-tools.cjs` (--force flag parsing + content-length guard), install sync to `~/.claude/qgsd/bin/gsd-tools.cjs`, and a passing `npm test` (148 total: 145 baseline + 3 new).
</objective>

**Change 2 — Add content-length guard inside `case 'regenerateState':` (before the backup block at line 3927):**

Current code starting at line 3927:
```javascript
case 'regenerateState': {
  // Create timestamped backup before overwriting

    const backupPath = `${statePath}.bak-${timestamp}`;

  }
  // Generate minimal STATE.md from ROADMAP.md structure
  ...
```

Replace so the case reads:
```javascript
case 'regenerateState': {
  // Content-length safety gate — prevent silent overwrite of rich STATE.md
  if (fs.existsSync(statePath)) {
    const existingContent = fs.readFileSync(statePath, 'utf-8');
    const lineCount = existingContent.split('\n').length;
    const SAFE_LINE_THRESHOLD = 50;
    if (lineCount > SAFE_LINE_THRESHOLD && !options.force) {
      repairActions.push({
        action: 'regenerateState',
        success: false,
        skipped: true,
        reason: `STATE.md has ${lineCount} lines (threshold: ${SAFE_LINE_THRESHOLD}). Re-run with --force to overwrite.`,
      });
      break;
    }
  }
  // Create timestamped backup before overwriting
  if (fs.existsSync(statePath)) {
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-').slice(0, 19);
    const backupPath = `${statePath}.bak-${timestamp}`;
    fs.copyFileSync(statePath, backupPath);
    repairActions.push({ action: 'backupState', success: true, path: backupPath });
  }
  // Generate minimal STATE.md from ROADMAP.md structure
  ...  (rest of the case is UNCHANGED — do not touch backup or template-write code)
```

**Critical constraints:**
- The guard goes INSIDE `case 'regenerateState':` only — do NOT add it to `createConfig` or `resetConfig` cases (pitfall from RESEARCH.md)
- `options.force` is only checked inside this case — do NOT check it at the outer repair-loop level
- The exit code remains 0 regardless (skipped case does not throw — pitfall from RESEARCH.md)
- Use `content.split('\n').length` (simple split) — not filter for empty lines
- The threshold is hardcoded as `const SAFE_LINE_THRESHOLD = 50` inside the case block (not a flag)
