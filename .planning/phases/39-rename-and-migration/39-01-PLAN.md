Create a new CommonJS script at `bin/migrate-to-slots.cjs`. It must implement:

**SLOT_MIGRATION_MAP** (hardcoded):
```javascript
const SLOT_MIGRATION_MAP = {
  'codex-cli':         'codex-cli-1',
  'gemini-cli':        'gemini-cli-1',
  'opencode':          'opencode-1',
  'copilot-cli':       'copilot-1',
  'claude-deepseek':   'claude-1',
  'claude-minimax':    'claude-2',
  'claude-qwen-coder': 'claude-3',
  'claude-kimi':       'claude-4',
  'claude-llama4':     'claude-5',
  'claude-glm':        'claude-6',
};
```

**`migrateClaudeJson(claudeJsonPath, dryRun)`** function:
- Read and JSON.parse the file
- For each `[oldName, newName]` in SLOT_MIGRATION_MAP:
  - If `servers[oldName]` exists AND `servers[newName]` does NOT exist → rename (assign + delete)
  - If `oldName` absent and `newName` present → already migrated (skip, idempotent)
  - If both present → skip (safety — don't overwrite)
- If `changed > 0` AND NOT dryRun → write back with `JSON.stringify(raw, null, 2) + '\n'`
- Return `{ changed, renamed: [{from, to}] }`

**`migrateQgsdJson(qgsdJsonPath, dryRun)`** function:
- Read and JSON.parse the file; if file does not exist, return `{ changed: 0 }`
- Look for `required_models` object; for each entry, check if `tool_prefix` matches old prefix pattern:
  - `'mcp__codex-cli__'`    → replace with `'mcp__codex-cli-1__'`
  - `'mcp__gemini-cli__'`   → replace with `'mcp__gemini-cli-1__'`
  - `'mcp__opencode__'`     → replace with `'mcp__opencode-1__'`
  - `'mcp__copilot-cli__'`  → replace with `'mcp__copilot-1__'`
- Count replacements; if `changed > 0` AND NOT dryRun → write back
- Return `{ changed, patched: [{ key, from, to }] }`

**CLI entrypoint** (when run directly via `require.main === module`):
- Parse `--dry-run` flag from `process.argv`
- Resolve `~/.claude.json` and `~/.claude/qgsd.json` paths using `os.homedir()`
- Call both migrate functions
- Print summary:
  - If dryRun: `[DRY RUN] Would rename N entries: old → new ...`
  - If nothing to migrate: `Already migrated — no changes needed`
  - Otherwise: `Migrated N mcpServers entries` + list + `Patched M qgsd.json tool_prefix values`
- Exit 0

The script must use only built-in Node.js modules (`fs`, `path`, `os`). No external dependencies.

Export `{ migrateClaudeJson, migrateQgsdJson, SLOT_MIGRATION_MAP }` for use by install.js.
