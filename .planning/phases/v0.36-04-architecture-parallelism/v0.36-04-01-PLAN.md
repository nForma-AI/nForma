<objective>
Collapse the 3-layer formal verification architecture (L1-L2-L3) to a 2-layer model (L1-L3) by removing the empty L2 (Semantics) layer. L2 has zero entries and permanently blocks Gate B at score 0. After this change, Gate A evaluates L1-to-L3 directly, Gate B is repurposed as a "purpose" check (L3 models have requirement backing), and Gate B scores above zero.
</objective>

<must_haves>
1. L2 (Semantics) layer is collapsed — the 2-layer architecture has Gate A evaluating L1-to-L3 directly, and Gate B no longer reports a permanent zero score
2. All consumers of LAYER_KEYS that reference l1_to_l2 and l2_to_l3 are updated to use the new key names (l1_to_l3 replaces l1_to_l2; l2_to_l3 is removed)
</must_haves>
