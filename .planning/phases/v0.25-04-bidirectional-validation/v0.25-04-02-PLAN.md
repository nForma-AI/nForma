Create `bin/check-coverage-guard.cjs` — a standalone CLI tool that compares the current traceability matrix coverage against a baseline and warns on regressions.

**Usage:**
```
node bin/check-coverage-guard.cjs                    # compare current vs baseline
node bin/check-coverage-guard.cjs --save-baseline    # save current matrix as baseline
node bin/check-coverage-guard.cjs --threshold 10     # override threshold (default: 15)
node bin/check-coverage-guard.cjs --quiet            # suppress output (exit code only)
```

**File paths:**
```javascript
const MATRIX_PATH   = path.join(ROOT, 'formal', 'traceability-matrix.json');
const BASELINE_PATH = path.join(ROOT, 'formal', 'traceability-matrix.baseline.json');
```

**`--save-baseline` mode:**
1. Read current `formal/traceability-matrix.json`
2. Copy it to `formal/traceability-matrix.baseline.json`
3. Print: `[check-coverage-guard] Baseline saved: {coverage_percentage}% coverage ({covered_count}/{total_requirements} requirements)`
4. Exit 0

**Comparison mode (default):**

1. Read `formal/traceability-matrix.json`. If missing, print error and exit 1.

2. Read `formal/traceability-matrix.baseline.json`. If missing:
   - Auto-create baseline from current matrix
   - Print: `[check-coverage-guard] No baseline found — creating from current matrix ({coverage_percentage}%)`
   - Print: `[check-coverage-guard] PASS: No regression (baseline just created)`
   - Exit 0

3. Extract coverage percentages:
   ```javascript
   const currentPct  = current.coverage_summary.coverage_percentage;
   const baselinePct = baseline.coverage_summary.coverage_percentage;
   const drop = baselinePct - currentPct;
   ```

4. Compare against threshold:
   - If `drop > threshold`:
     ```
     [check-coverage-guard] WARN: Coverage regression detected
     [check-coverage-guard]   Baseline: {baselinePct}% ({baseline.coverage_summary.covered_count}/{baseline.coverage_summary.total_requirements})
     [check-coverage-guard]   Current:  {currentPct}% ({current.coverage_summary.covered_count}/{current.coverage_summary.total_requirements})
     [check-coverage-guard]   Drop:     {drop} percentage points (threshold: {threshold}pp)
     [check-coverage-guard]   To update baseline: node bin/check-coverage-guard.cjs --save-baseline
     ```
     Exit 1

   - If `drop > 0` but `drop <= threshold`:
     ```
     [check-coverage-guard] PASS: Minor coverage change ({drop}pp drop, within {threshold}pp threshold)
     [check-coverage-guard]   Baseline: {baselinePct}% → Current: {currentPct}%
     ```
     Exit 0

   - If `drop <= 0` (coverage improved or unchanged):
     ```
     [check-coverage-guard] PASS: No regression (baseline: {baselinePct}% → current: {currentPct}%)
     ```
     Exit 0

**CLI flag parsing:**
```javascript
const args = process.argv.slice(2);
const saveBaseline = args.includes('--save-baseline');
const quietMode    = args.includes('--quiet');
const thresholdIdx = args.indexOf('--threshold');
const threshold    = thresholdIdx !== -1 ? parseFloat(args[thresholdIdx + 1]) : 15;
```

**Style notes:**
- Use `#!/usr/bin/env node` shebang and `'use strict'`
- TAG constant: `const TAG = '[check-coverage-guard]'`
- Use `process.stdout.write(TAG + ' ...\n')` for output (matching project pattern)
- Use `process.stderr.write()` for error messages
- Round `drop` to 1 decimal place for display: `Math.round(drop * 10) / 10`
