## Deviations from Plan
1. **createActor source**: Plan specified `require('xstate')` but xstate is a devDependency not available at `~/.claude/qgsd-bin/`. Instead, `createActor` is re-exported from the machine CJS (which bundles xstate via tsup). This makes the validator self-contained.
2. **Dual-path resolution for machine CJS**: Plan specified a fixed relative path `../dist/machines/`. Implemented dynamic path resolution (repo path first, install path fallback) to support both development and installed contexts.
3. **Manual dist/machines copy**: The installer's `bin/*.cjs` loop doesn't handle `dist/machines/` subdirectory. The machine CJS was manually copied to `~/.claude/qgsd-bin/dist/machines/` for this phase. A future plan should update install.js to handle this automatically.
