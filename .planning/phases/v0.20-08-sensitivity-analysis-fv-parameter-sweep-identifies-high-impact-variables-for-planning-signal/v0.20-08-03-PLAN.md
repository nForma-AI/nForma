<feature>
  <name>sensitivity-report.cjs — Ranked Sensitivity Report Generator</name>
  <files>bin/sensitivity-report.cjs</files>
  <behavior>
    Reads `formal/sensitivity-report.ndjson` (or SENSITIVITY_REPORT_PATH env var).
    Groups records by `metadata.parameter`.
    Counts `metadata.delta === 'flip-to-fail'` or `'flip-to-inconclusive'` per parameter.
    Ranks parameters by flip count (descending).
    Writes `formal/sensitivity-report.md` (or SENSITIVITY_MD_PATH env var).

    **Hardcoded annotations** (code paths known from codebase analysis):
    ```javascript
    const PARAM_ANNOTATIONS = {
      MaxSize: {
        codePath: 'hooks/qgsd-prompt.js FAN_OUT_COUNT; formal/tla/MCsafety.cfg MaxSize',
        testCases: [
          'Test quorum at N=2 boundary: set FAN_OUT_COUNT=2 in providers.json and run quorum',
          'Test quorum at N=1 (no quorum): verify workflow rejects insufficient slots',
        ],
        monitoring: [
          'Track FAN_OUT_COUNT per planning session in quorum-scoreboard.json',
          'Alert if available slot count drops below MaxSize threshold',
        ],
      },
      tp_rate: {
        codePath: 'bin/export-prism-constants.cjs TP_PRIOR=0.85; formal/prism/quorum.pm tp_rate',
        testCases: [
          'Test quorum with 2/4 slots returning APPROVE (tp_rate≈0.5) — verify inconclusive behavior',
          'Test quorum with all slots UNAVAILABLE — verify graceful degradation',
        ],
        monitoring: [
          'Track per-slot APPROVE rate in quorum-scoreboard.json rounds',
          'Alert if any slot tp_rate falls below 0.6 for 5+ consecutive rounds',
        ],
      },
    };
    ```

    **Report format**:
    ```markdown
    # Sensitivity Report — QGSD v0.20

    Generated: {ISO timestamp}
    Source: formal/sensitivity-report.ndjson ({N} records)

    ## High-Sensitivity Parameters (ranked by outcome-flip count)

    ### 1. {param.name} — {N} flips
    **Description:** {param description}
    **Flip values:** {list of values that caused flips}
    **Code path:** {codePath annotation}

    **Recommended test cases:**
    - {test case 1}
    - {test case 2}

    **Recommended monitoring:**
    - {metric 1}
    - {metric 2}

    ---
    ```

    If no flips detected (all results stable or all inconclusive):
    ```markdown
    ## No Outcome Flips Detected

    All sweep results were inconclusive (tools not installed) or stable.
    Install TLC and PRISM to run a live sensitivity sweep.
    ```

    Always exits 0. Fail-open if input file missing (writes "No data available" report).
  </behavior>
</feature>
