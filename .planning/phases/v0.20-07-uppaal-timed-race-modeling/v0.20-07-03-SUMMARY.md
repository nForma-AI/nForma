## Model structure

**quorum-races.xml** (3 templates):
- `SlotWorker`: clock x, Waiting (invariant x<=MAX_SLOT_MS) → Responded (guard x>=MIN_SLOT_MS, sync response!)
- `QuorumOrchestrator`: clock t, Collecting (invariant t<=TIMEOUT_MS) → Decided (v>=MIN_QUORUM) or TimedOut (t>=TIMEOUT_MS)
- `RaceDetector`: Idle → Counting (response?) → Race (second simultaneous response)

**quorum-races.q** (4 queries):
```
A[] not deadlock          // UPPAAL-01: no deadlock
E<> orch.Decided          // UPPAAL-03a: consensus reachable
E<> orch.Decided and orch.t < TIMEOUT_MS  // UPPAAL-03b: before deadline
A[] not race.Race         // UPPAAL-03a: race-free
```
