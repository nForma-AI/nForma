  <name>Task 1: Create prism-priority.cjs -- PRISM failure priority ranker</name>
  <files>bin/prism-priority.cjs</files>
  <action>
Create `bin/prism-priority.cjs` following existing QGSD patterns (CommonJS, 'use strict').

1. **Impact score map:**
   ```js
   const IMPACT_SCORES = {
     'prism:quorum': 10,
     'prism:mcp-availability': 7,
   };
   const DEFAULT_IMPACT = 5;
   ```

2. **`readPrismResults(ndjsonPath)`** function:
   - Reads check-results.ndjson (default: `$CWD/formal/check-results.ndjson`).
   - Filters for entries where `formalism === 'prism'`.
   - For each PRISM entry, extracts: `check_id`, `result`, `summary`, `timestamp`, `metadata`.
   - Groups by check_id, keeping only the most recent entry per check_id.
   - Returns array of `{ check_id, result, summary, timestamp, metadata }`.
   - If file doesn't exist, returns empty array.

3. **`rankFailureModes(prismResults)`** function:
   - Takes array from readPrismResults.
   - Filters for entries where `result === 'fail'` or `result === 'warn'`.
   - For each failure mode:
     - Extract P(failure) from metadata or summary. For fail results, P(failure) = 1.0. For warn results with cold-start mode, P(failure) = 0.5 (uncertain).
     - Look up impact score from IMPACT_SCORES map.
     - Compute priority = P(failure) x impact.
   - Sort descending by priority.
   - Returns array of `{ check_id, priority, p_failure, impact, summary }`.
   - If no failures: returns empty array.

4. **`formatPrioritySignal(rankedModes)`** function:
   - Takes output of rankFailureModes.
   - If empty: returns null.
   - Produces formatted text block:
     ```
     === PRISM Priority Signal ===
     Failure modes ranked by P(failure) x impact:
     1. [prism:quorum] priority=10.0 -- P(fail)=1.0, impact=10 -- "fail: quorum in 500ms"
     2. [prism:mcp-availability] priority=3.5 -- P(fail)=0.5, impact=7 -- "warn: mcp-availability in 200ms"
     =============================
     ```
   - Returns the block as a string.

5. **CLI entrypoint** (guarded by `require.main === module`):
   - Usage: `node bin/prism-priority.cjs [--path=check-results.ndjson]`
   - Reads, ranks, formats. Prints to stdout.
   - If no failures: prints "No PRISM failures found" to stderr and exits 0.

6. **Exports:** `module.exports = { readPrismResults, rankFailureModes, formatPrioritySignal }`.
