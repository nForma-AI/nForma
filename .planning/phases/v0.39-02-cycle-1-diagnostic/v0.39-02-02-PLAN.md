<objective>
Reframe all user-facing output from "bug not captured" to "model is incomplete/wrong" across the refinement loop, model-driven-fix workflow, and close-formal-gaps workflow.

Purpose: The bug is ground truth (the oracle). The model is the hypothesis under test. Current language implies the bug description might be wrong ("bug not captured"); correct language makes clear the model is what needs improvement ("model is incomplete"). This reframing is critical for guiding users toward model improvement rather than bug re-description.
Output: Updated strings in refinement-loop.cjs and workflow markdown files; updated test expectations.
</objective>
