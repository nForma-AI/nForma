# Summary: Config Write Adapter

## What Was Built
Added a write-side config normalization adapter to config-loader.js. Boolean strings are converted to booleans, profile/tier/fail_mode values are lowercased, and nested key formats like `{ model_tier: { planner: 'opus' } }` are bidirectionally converted to/from flat keys like `model_tier_planner`. A writeConfig function produces properly normalized JSON output.
