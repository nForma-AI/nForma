    Hook control flow (in order):
    1. Parse stdin JSON -- extract input.cwd and input.tool_input.command
    2. Resolve git root via execSync('git rev-parse --show-toplevel', {cwd, stdio:'pipe', timeout:5000})
       -- if throws (non-git dir or error): exit 0 immediately (DETECT-05)
    3. Read state file at path.join(gitRoot, '.claude', 'circuit-breaker-state.json')
       -- if state.active === true: exit 0 (Phase 6 pass; Phase 7 will block here)
    4. Check isReadOnly(command) -- if true: exit 0 (DETECT-04)
    5. Get last 6 commit hashes via execSync('git log --format="%H" -6', {cwd:gitRoot,...})
    6. For each hash: execSync('git diff-tree --no-commit-id -r --name-only HASH', {cwd:gitRoot,...})
       -- split on newline, filter empty -- produces per-commit file array
    7. Oscillation check: sort each file array, join with '\0', count occurrences via Map
       -- if any key count >= 3: oscillation detected (DETECT-03, strict set equality)
    8. If oscillation: write state file { active:true, file_set:sortedFiles, activated_at:ISO,
       commit_window_snapshot:allPerCommitArrays } -- mkdirSync recursive, writeFileSync (STATE-01/02/04)
    9. exit 0 (Phase 6 always passes even after writing state)

    Read-only regex: /^\s*(git\s+(log|diff|diff-tree|status|show|blame)|grep|cat\s|ls(\s|$)|head|tail|find)\s*/

    State file location: path.join(gitRoot, '.claude', 'circuit-breaker-state.json')
    State schema: { active: bool, file_set: string[], activated_at: string, commit_window_snapshot: string[][] }
